# paper-anime

ペーパークラフトで作るアニメーション

## 概要

このプロジェクトは、ペーパークラフトを使ってゾートロープ（回転のぞき絵）を作成し、アニメーションを楽しむためのリソースを提供しています。

## フィルムメーカーアプリ

Scratchベースのフィルムメーカーアプリを使って、オリジナルのアニメーションフィルムをデザインできます。

🎬 [フィルムメーカーアプリを開く](https://xcratch.github.io/editor/#https://tukumanalab.github.io/paper-anime/app/film-maker.sb3)

## ペーパークラフトデータ

MFT2025向けのゾートロープペーパークラフトデータは以下からダウンロードできます：

### データファイル

- [ゾートロープのつくりかた (PDF)](data/zoetrope-mft2025/ゾートロープのつくりかた-MFT2025.pdf)
- [ペーパークラフト本体 (PDF)](data/zoetrope-mft2025/zt-papercraft-mft2025.pdf)
- [フィルム用紙 (PDF)](data/zoetrope-mft2025/zt-film-mft2025.pdf)
- [Helix版ペーパークラフト (PDF)](data/zoetrope-mft2025/zt-papercraft-mft2025-helix.pdf)
- [Fabool版ペーパークラフト (SVG)](data/zoetrope-mft2025/zt-papercraft-mft2025-fabool.svg)

### その他のフォーマット

- SVGファイル（レーザーカッター用）
- DXFファイル（CAD用）

すべてのファイルは [`data/zoetrope-mft2025`](data/zoetrope-mft2025/) ディレクトリにあります。

## つくりかた

1. PDFファイルをダウンロードして印刷
2. 型紙に沿って切り抜き
3. 組み立て
4. フィルムメーカーアプリでアニメーションを作成
5. フィルムを印刷してゾートロープにセット
6. 回転させて楽しむ！

詳しい作り方は「ゾートロープのつくりかた」PDFをご覧ください。
